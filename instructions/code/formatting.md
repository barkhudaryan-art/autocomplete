# Formatting

All code must pass `eslint.config.mjs`.

- Tabs (not spaces).
- Single quotes, no trailing commas, semicolons always.
- Spaces inside parens: `if ( x )`, `fn( arg )`, `useState( v )`.
- No spaces in array brackets: `[1, 2, 3]`.
- Spaces in object braces: `{ key: value }`.
- No space before function parens: `function name()`, `myFn()`.
- 1TBS, curly braces always.
- 2 blank lines after imports (`import/newline-after-import: 2`).
- 1 blank line before/after function and class declarations.
- No trailing blank line at EOF.
- Max line length: 150 (warn).
- No trailing spaces; max 2 consecutive blank lines.
